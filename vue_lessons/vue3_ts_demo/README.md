# Vue3 Typescript Demo

## Build Project

1. Create a new vue3 project.

```sh
npm i -g @vue/cli
vue --version

# select: Manually select features, enable: typescript, router
vue create vue3_ts_demo

# run
cd vue3_demo_basic
npm run serve
```

2. Edit config: `tsconfig.json` and `vue.config.js`
